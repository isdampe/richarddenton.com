Things

Here's some web related things I work on in my spare time.

Boson

Boson is a minimalistic experimental source code editor. It's written in Node, and wrapped in Nw.js as a runtime container for easy cross-platform integration. It supports theming, plugins, multi-pane editing, and works on Windows, Linux and OSX.

Boson on Github

Notedown

Notedown is a cross-platform desktop application, that allows you to keep notes in the popular web syntax Markdown.

It's built with HTML5 / CSS3 / JS, and is wrapped in the node-webkit chromium wrapper.

It's free software, released under the MIT license.

Notedown on Github

PrettyPress

PrettyPress is a Wordpress plugin that provides a modern post editor for established and fast work flow.

As well as a markdown editor, it also provides the user a live preview of their post while they type.

PrettyPress on Github - PrettyPress on Wordpress.org
